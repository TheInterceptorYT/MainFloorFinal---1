# The Second Puzzle

# Description

After you complete the kitchen puzzle and narrowly escape the rampant animatronics you will come across the main floor. This is the second puzzle of the game FNAF Return of the Souls, built by me!

# Instructions

In order to move past this puzzle you must first look at the Fazbear Entertainment Poster left behind. it's a hint showing the order in which the food must be placed in once you click on the plate. Don't take too long or Freddy will get ya!

# Features

An interactive puzzle
Custom animations
Music and Sound Effects
